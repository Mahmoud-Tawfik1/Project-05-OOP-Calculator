# 🧮 OOP Calculator (C++)

A console-based **Calculator** built using **C++** to demonstrate the fundamentals of **Object-Oriented Programming (OOP)**.

This project focuses on applying **Encapsulation** and **Abstraction** through a reusable calculator class capable of performing arithmetic operations while maintaining its internal state.

## 🎯 Project Objectives

- Practice Object-Oriented Programming in C++
- Apply Encapsulation and Abstraction
- Design reusable classes and member functions
- Manage object state through private data members
- Build a maintainable calculator application

---

## ✨ Features

🔹 Addition

🔹 Subtraction

🔹 Multiplication

🔹 Division with basic validation

🔹 Automatic result tracking

🔹 Store previous operation information

🔹 Undo the last operation

🔹 Reset calculator state

🔹 Display the current and final results

## ⚙️ Calculator Operations

The calculator supports:

- Add
- Subtract
- Multiply
- Divide
- Cancel Last Operation
- Clear Calculator
- Print Current Result
- Retrieve Final Result

Each operation updates the calculator object while keeping implementation details hidden from the user.
